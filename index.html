Instructions

To download your followers and following, please follow the instructions below.

  1. Go to your Meta Account Center
  2. Navigate to Your Information
  3. Select Download Your Information
  4. Click Request Download
  5. Select the Instagram account you want to compare
  6. Selecting "Select types of information" is faster
    1. If you select this, select Followers and Following
  7. Submit the Request
  8. Wait for the link to arrive by email (This may take some time)
  9. Once you receive the email, follow the link
  10. Download the file
  11. Unzip the file
  12. Open the folder, and navigate to connections/followers_and_following folder
  13. Select the followers_1.html and following.html file with this tool
